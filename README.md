# FYP Showcase

## Introduction
FYP Showcase is a simple static website developed as an academic project. The purpose of this website is to provide a platform where students can showcase their Final Year Projects (FYPs) and related information in an organized manner.

The website includes multiple pages such as Home, About, Services, and Contact. It demonstrates the basic use of HTML and CSS for layout, styling, and navigation. Along with the website, a theoretical SQL database design is included to represent how data such as users, services, and contact messages can be stored and managed.

## Objectives
The main objectives of this project are:
* To understand the structure of a multi-page website
* To practice HTML and CSS for designing web pages
* To learn basic database concepts using SQL
* To design and use simple visuals using Canva
* To upload and manage a project using GitHub

## Tools Used
The following tools and technologies were used to complete this project:
* **HTML:** For creating the structure of web pages
* **CSS:** For styling and layout of the website
* **Visual Studio Code:** As the main code editor
* **SQLite (Theoretical):** For database design and SQL queries
* **Canva:** For creating the website banner, logo, and poster
* **GitHub:** For hosting and sharing the project files

## Website Pages Explanation

### Home Page
The Home page serves as the main landing page of the website. It includes a banner section, navigation menu, featured student projects, and a poster section. This page gives an overview of the platform and highlights important updates.

### About Page
The About page explains the purpose and mission of the FYP Showcase website. It also introduces the team members involved in the project and demonstrates content alignment, images, tables, and text formatting.

### Services Page
The Services page displays the features or services offered by the platform. These services are related to showcasing student projects, providing information, and supporting academic presentation.

### Contact Page
The Contact page contains a form that allows users to submit their name, email, and message. This page represents how user input can be collected and stored in a database. It also includes address information formatted using pre-formatted text.

## Canva Designs
Canva was used to design visual elements for the website. These include:
* Website banner used on the Home page
* Logo representing the FYP Showcase platform

These designs help improve the visual appearance of the website and make it more attractive for users.
